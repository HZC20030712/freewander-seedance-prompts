## Acceptance Rules

Only the following source license groups may enter Phase 1:

- MIT
- Apache 2.0
- BSD 2-Clause / 3-Clause
- CC0
- CC BY 4.0

The following source license groups are excluded from Phase 1:

- CC BY-NC
- CC BY-ND
- GPL
- AGPL
- no license

## Attribution Requirement

Every prompt entry must carry source metadata, including:

- source repo name
- source repo URL
- original author
- original license
- collected timestamp
- last sync timestamp
