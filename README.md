## Quick Start

1. Browse the manifest in `data/index.json`
2. Open category files under `data/seedance/`
3. Copy prompts into FreeWander or your own workflow

```bash
git clone https://github.com/HZC20030712/freewander-seedance-prompts.git
cd freewander-seedance-prompts
```
